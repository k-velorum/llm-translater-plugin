# アイコンファイルについて

Chromeの拡張機能には以下の3つのサイズのアイコンが必要です：

- 16x16ピクセル (icon16.png)
- 48x48ピクセル (icon48.png)
- 128x128ピクセル (icon128.png)

## アイコンの作成方法

1. お好みの画像エディタ（PhotoshopやGIMP、Figmaなど）を使用して、各サイズのアイコンを作成してください
2. 作成したアイコンを以下のファイル名で保存してください：
   - `icon16.png` (16x16ピクセル)
   - `icon48.png` (48x48ピクセル)
   - `icon128.png` (128x128ピクセル)
3. 作成したアイコンファイルをこのディレクトリ（`icons`フォルダ）に配置してください

## アイコン作成のヒント

- シンプルで認識しやすいデザインを心がけましょう
- 小さいサイズ（特に16x16）でも識別できるようにしましょう
- 透過PNGを使用すると、さまざまな背景に適応します
- 翻訳や言語を象徴する要素を含めると良いでしょう（例：吹き出し、地球儀、「翻訳」の文字など）

## オンラインツール

以下のオンラインツールを使用してアイコンを作成することもできます：

- [Favicon.io](https://favicon.io/) - テキストや画像からアイコンを生成
- [Icons8](https://icons8.com/) - 無料アイコンライブラリ
- [Canva](https://www.canva.com/) - オンラインデザインツール

これらのツールを使用して作成したアイコンをダウンロードし、適切なサイズにリサイズしてからこのディレクトリに保存してください。
